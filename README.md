# PolynomialCalculator
## Калькулятор полиномов, написанный на yacc (bison)
Поддерживаются следующие операции:
1) сложение, вычитание
```
(2x^2 + 3x - 8) - (-5x^2 - 10)
```
2) умножение
```
y * (y^2 + 5) * (y - 3)
```
3) возведение в степень
```
(x + 1)^5
```
4) использование переменных
```
$A = 5y
$B = y^10 - 20
($A * $B)^2 - $B
```
